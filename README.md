💼 Job Assistant AI
📌 Overview

Job Assistant AI is an intelligent application designed to help users with job-related tasks such as resume guidance, job recommendations, interview preparation, and career advice. It leverages AI capabilities to provide smart, context-aware responses and improve job search efficiency.

🎯 Features
🔍 Job recommendations based on user input
📄 Resume analysis and suggestions
💬 AI-powered chatbot for career guidance
🎤 Interview question preparation
📊 Personalized job insights
🌐 Integration with APIs (OpenAI / Groq / Hugging Face)
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript (or React if used)
Backend: Python (Flask)
AI Integration: OpenAI API / Groq API / Hugging Face
Database: MySQL (if used)
Other Tools: REST APIs, JSON
